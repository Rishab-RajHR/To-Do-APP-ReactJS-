A simple and interactive Todo List application built using React.js

Allows users to add, edit, delete, and mark tasks as completed

Uses React functional components and Hooks (useState) for state management

Provides a clean and responsive user interface

Demonstrates core React concepts like components, props, state, and event handling

Helps users stay organized by managing daily tasks efficiently

Easy to extend with features like local storage, filters, or due dates

Ideal for beginners to understand CRUD operations in (React)
